The videos are the demo of the app and application usage
The codes are made by using Arduino IDE, with following libraries being used:
- ArduinoHTTPClient
- Arduino_ESP32_OTA
- Arduino_JSON
- Adafruit BusIO
- Adafruit SSD1306
- ArduinoJson
- Async TCP
- ESP Async WebServer
- ESP32 Servo
- ESP32Servo360
- LiquidCrystal I2C
- MFRC522-spi-i2c-uart-async
- U8g2
- esp32cam
